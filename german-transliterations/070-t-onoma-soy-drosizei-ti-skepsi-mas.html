70 T' ONOMA SOY DROSIZEI TI SKEPsI MAS

Griechische Hymnen - Nr. 70 | "Τ' ΟΝΟΜΑ ΣΟΥ ΔΡΟΣΙΖΕΙ ΤΗ ΣΚΕΨΗ ΜΑΣ"
Verse 1:
T' onoma Sou drosizei ti skepsi mas,
(Dein Name kühlt unsere Gedanken,)
t' onoma Sou pigi eflogias,
(Dein Name ist eine Quelle des Segens,)
t' onoma Sou einai xypnima sto fos!
(Dein Name erwacht im Licht!)
T' onoma Sou mas dinei tin eisodo
(Ihr Name verschafft uns Zutritt)
sta ourania mprosta ston Patera!
(im Himmel vor dem Vater!)
T' onoma Sou yposchetai polla,
(Dein Name verspricht viel,)
Iisoys, afto einai t' onoma Sou!
(Jesus, das ist Dein Name!)
Chorus:
Thafmastos eisai Symvoulos, eisai Kyrios eirinis,
(Wunderbar, dass Sie Berater sind, Sie sind Herr des Friedens,)
o Pateras tou mellontos, o Theos tis galinis!
(der Vater der Zukunft, der Gott des Friedens!)
Ta paidia Sou tha psalloune, Iisoy, sto onoma Sou,
(Deine Kinder werden singen, Jesus, in Deinem Namen,)
kathe mera tha S' eflogoyn tapeina sti thoria Sou!
(Jeden Tag werden sie Dich in Deiner Gegenwart demütig segnen!)
Verse 2:
T' onoma Sou spathi einai sta cheili mas,
(Dein Name ist ein Schwert auf unseren Lippen,)
t' onoma Sou pigi eftychias,
(Dein Name Quelle des Glücks,)
t' onoma Sou einai kastro dynato!
(Dein Name ist eine mächtige Burg!)
T' onoma Sou anasa ki elpida mas,
(Dein Name ist unser Atem und unsere Hoffnung,)
t' onoma Sou proini iliachtida,
(Dein Name, Morgensonnenstrahl,)
t' onoma Sou einai minyma zois.
(Dein Name ist eine Botschaft des Lebens.)
Iisoys, afto einai t' onoma Sou!
(Jesus, das ist Dein Name!)
Verse 3:
T' onoma Sou pigi agiotitas,
(Dein Name ist eine Quelle der Heiligkeit,)
t' onoma Sou einai fos ki odigia,
(Dein Name ist Licht und Belehrung,)
t' onoma Sou Theos alithinos!
(Dein Name ist wahrer Gott!)
T' onoma Sou gemizei to einai mas,
(Dein Name erfüllt unser Wesen,)
t' onoma Sou dialyei to fovo,
(Dein Name vertreibt die Angst,)
t' onoma Sou einai agapi kai zoi,
(Dein Name ist Liebe und Leben,)
Iisoys, afto einai to onoma Sou!
(Jesus, das ist Dein Name!)
