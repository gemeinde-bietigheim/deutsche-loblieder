620. ΘΑ ΗΘΕΛΑ ΝΑ ΗΜΟΥΝΑ ΣΤΗ ΓΗ

Ελληνικοί Ύμνοι - Αρ. 620
Στροφή 1:
Θα ήθελα να ήμουνα στη γη,
όταν, Ιησού μου, έζησες Εσύ!
Θα ήθελα σαν τη Μαγδαληνή
να Σ' έβλεπα έστω και μια στιγμή!
Αχ! να 'μουνα να έβλεπα κι εγώ
τα θαύματα που έκανες εδώ,
να Σ' άκουγα να λες: «Σε συγχωρώ»,
να ήμουνα κοντά Σου στο Σταυρό!
Επωδός:
Κάποια μέρα σαν φτάσω εκεί στον ουρανό,
το δρόμο μου όταν τελειώσω κάτω εδώ
κι εκεί πέρα το πρόσωπό Σου σαν θα δω,
το όνειρό μου πια θα είν' αληθινό!
Στροφή 2:
Θα ήθελα να ήμουνα στη γη,
όταν, Ιησού μου, έζησες Εσύ!
Να Σ' άκουγα να λες για ουρανό,
για έναν τόπο τόσο αληθινό!
Να Σ' έβλεπα τα μάτια των τυφλών
να ανοίγεις μ' ένα λόγο σου γλυκό!
Να χόρταινα με ψάρι και ψωμί,
μια μέρα που τα ευλόγησες Εσύ!
